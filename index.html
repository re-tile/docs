MDE Software Documentation:

The MDE software documentation includes the documents listed below. When you work with a new release of the MDE, you should first read the MDE Release Notes and then Getting Started with the MDE.

Documents Supplied with the MDE Description
Getting Started with the Multicore Development Environment, UG504 This guide offers a brief introduction to the Multicore Development Environment™ (MDE) and provides instructions to help developers install the software and get up and running quickly using the IDE or the command-line interface. It is intended for use by individuals who are responsible for installing software and by application developers.
Programming the TILE-Gx Processor, UG505 This guide describes the basic concepts involved in developing applications for the Tile Processor™ and also provides specific implementation details. It is intended as a next step for application developers who have worked through the Getting Started Guide (UG504). We also recommend that you read the Tile Processor Architecture Overview for the TILE-Gx Series (UG130).
MDE User Guide, UG501 This guide describes how to use the MDE, providing instructions for creating, running, debugging, and profiling applications using the command-line tools. It is intended for use by application developers.
Using the Tilera IDE, UG511 This guide describes how to use the tile-eclipse Integrated Development Environment (IDE), a version of the standard Eclipse graphical IDE enhanced for use in programming Tilera's multicore processors. It is intended for use by application developers.
MDE System Programmer's Guide, UG509 This guide describes how to customize the system software environment for the Tile Processor. It is intended for use by developers and administrators responsible for the OS, device drivers, and other software running at the system level.
MDE Optimization Guide, UG515 This guide describes how to improve the performance of multicore applications that are using the MDE. It is intended for use by application developers.
Migrating to the TILE-Gx Processor Family, UG521 This guide describes how to migrate applications from TILEPro and previous MDE versions to TILE-Gx and MDE 4.0. It is intended for use by application and system developers.

TILE-Gx Processor Architecture

This section lists hardware documentation for the TILE-Gx series processors. The books listed in this section are divided into Architecture and Platform documentation.

Documents Supplied with the MDE Description
Architecture Manuals
Tile Processor Architecture Overview for the TILE-Gx Series, UG130 This manual provides an overview of the TILE-Gx architecture, identifies the features, briefly identifies the supported interfaces and methods for using them.
TILE-Gx Instruction Set Architecture Specification, UG401 (or PDF) This manual describes the Instruction Set Architecture (ISA) for the TILE-Gx Processor.
TILE-Gx Special Purpose Register Specification This manual describes the Special Purpose Registers of the TILE-Gx Processor.
Tile Processor and I/O Device Guide, UG404 This manual describes the Tile-Gx Processor's system architecture and its I/O interfaces. See also I/O Device Documentation below.
Platform Manuals
TILEmpower-Gx36 Platform User's Guide, UG415 This guide describes usage of Tilera's TILEmpower-Gx36™.
TILEmpower-Gx FR Platform User's Guide, UG419 This guide describes usage of Tilera's TILEmpower-Gx FR™.
TILEmpower-Gx72 MP Platform User's Guide, UG420 This guide describes usage of Tilera's TILEmpower-Gx72 MP™.
TILEncore-Gx36 Card User's Guide, UG416 This guide describes usage of Tilera's TILEncore-Gx36™.
TILEncore-Gx72 Card User's Guide, UG414 This guide describes usage of Tilera's TILEncore-Gx72™.
TILEncore-Gx Intelligent Application Adapter User's Guide, UG417 This guide describes usage of the TILEncore-Gx Intelligent Application Adapter, featuring the TILE-Gx16 or TILE-Gx9 processors.
TILEncore-Gx36 Q40 Card User's Guide, UG421 This guide describes usage of the TILEncore-Gx36 Q40 PCIe card.
Baseboard Management Controller User's Guide, UG533 This guide describes usage of Tilera's Baseboard Management Controller.
Tilera Board Test Kit Guide for TILE-Gx Processors, UG422 This guide explains how to use the Board Test Kit (BTK) to run a set of tests on a TILE-Gx based platform, as well as how to develop additional tests using the unique features of the platform.
