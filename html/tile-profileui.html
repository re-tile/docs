Name

tile-profileui — Runs the Profile UI, a "dashbord" for performing profiling runs on TILE hardware and viewing the results.

Synopsis

tile-profileui [-h | --help]

Description

The Profile UI, invoked by the tile-profileui command, is a "dashboard"-style front-end for the profiling utilities included in the MDE (at present, OProfile and Linux perf events).

Command-Line Options

-h | --help

Optional. Displays usage information and list of command options, then exits.

Usage

To start the Profile UI, invoke the command tile-profileui. This displays the Profile UI main window, a "dashboard" style control panel providing access to the profiling tools of the MDE.

The Profile UI allows you to configure and start/stop the profiling tools. It also allows you to invoke a complete init/start/profile/stop/capture run for your application as a single button-click.

For inspecting the resulting captured profiling data, the Profile UI provides a number of single-click views, including aggregating data by process/thread and by function name, plus caller/callee info, cpu usage, and more.

For more information on the Profile UI and its features, please see the online help, accessed via the Help->Help menu command in the Profile UI itself.
