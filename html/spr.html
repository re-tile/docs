    Data Stream Prefetch Control (DSTREAM_PF: 0x2706)
    This register controls hardware initiated Dstream prefetches. After a load instruction, the prefetcher may be armed to request a non-faulting prefetch request at the address of the load plus the value in the Stride field. Prefetches will not cross a 16kB boundary.
    Bits Name Type Reset Description
    18 MISS_ONLY RW 0 Controls whether or not the prefetcher is armed after a load instruction based on whether or not the load hits in L1 Dcache. 0 = independent of L1 Dcache hit, 1 = only after L1 Dcache miss.
    17:16 LEVEL RW 0 Which cache to allocate for dstream prefetches. 0 = L1 Dcache, 1 = L2 Cache, 2 or 3 = L3 Cache.
    12:6 STRIDE RW 0 The amount added to the load address for a stream using STRIDE mode. In the MODE field is STRIDE, a value of 0 disables prefetching. In other modes, a non-zero value puts one stream into STRIDE mode.
