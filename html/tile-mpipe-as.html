Name

tile-mpipe-as — Assembler for TILE-Gx Classifier Processor

Synopsis

tile-mpipe-as -o output_file [input_file]

Description

tile-mpipe-as assembles classifier programs.

Classifier Programs

tile-mpipe-as is normally called by tile-mpipe-cc, but may also be invoked directly from the command line.

[ Note: for complete details of the Classifier and its instruction set, please see the Tile Processor I/O Device Guide for the TILE-Gx Family of Processors. ]

Options

-o output_file

output_file is required. It is the filename of the executable classifier program written by tile-mpipe-as.

input_file

input_file is optional. It is the filename of the classifier program source. If input_file is omitted, the standard input will be used instead.
