3.17.37 TILEPro Options

These ‘-m’ options are supported on the TILEPro:

-m32
Generate code for a 32-bit environment, which sets int, long, and pointer to 32 bits. This is the only supported behavior so the flag is essentially ignored.
-mbme
Compile using the headers and libraries for Bare Metal Environment (BME).
-mcpu=name
Selects the type of CPU to be targeted. Currently the only supported type is ‘tilepro’.
-mexpand-asm
Expand certain inline assembly into rtl patterns that the compiler can better optimize. It's on by default and is used to support post incrementing memory intrinsics.
-mnewlib
Compile using the newlib headers and libraries.
-march=name
-momit-leaf-frame-pointer
-mtune=name
These flags are ignored, and are here for x86 compatibility.
-Wunmatched-loop-pragmas
Warn about loop pragmas that do not have matching loops.
