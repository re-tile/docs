Name

tile-gen-initramfs — Create an initramfs.cpio file for a bootrom

Synopsis

tile-gen-initramfs [option...] {contents} {initramfs.cpio}

Description

This program creates a Linux-format initramfs.cpio file, suitable for including in the hypervisor filesystem (or directly embedding in the vmlinux file itself). It will automatically strip Elf-format files read from the root filesystem. The program functions as a wrapper around the gen_init_cpio binary that comes as part of the Linux source distribution.

The input to the program is in the form of a "contents" file which lists the files, directories, symlinks, and device nodes to be included in the root RAM filesystem of the booted kernel. The file format is documented by the gen_init_cpio program itself when run without arguments, and a sample such file is present as /usr/lib/initramfs/contents.txt in the Tile filesystem hierarchy ($TILERA_ROOT/tile in the cross-build environment).

There are several extensions to the existing gen_init_cpio program provided by tile-gen-initramfs.

First, the "location" of the file, which is normally specified as a path in the current filesystem, is extended by allowing two special prefixes, "ROOT" and "DIR". The "ROOT" prefix is equivalent to "${TILERA_ROOT}/tile" in the cross-build environment, or "" (i.e. empty) in the native environment. The "DIR" prefix is expanded to the directory portion of the specified contents file path, making it easy to put references in the "contents" file to files stored in the same directory.

Second, the "ROOT" files are checked prior to being added to the cpio archive, and if they are Elf files, they are by default stripped, to save space in the initramfs image. The --strip-all option removes all symbols from the binaries including those that are otherwise available to the dynamic loader. The --strip-debug removes only the debugging symbols. Finally, the --no-strip causes no stripping to be done at all.

Finally, the output file may be specified as ending with ".cpio" to create a standard Linux initramfs.cpio file, or you may specify ".cpio.gz", ".cpio.bz2", or ".cpio.xz" to compress the generated file with the specified compression type.

Options

--strip-all

Remove all symbols from files starting with "ROOT/".

--strip-debug

Remove only debugging symbols from files starting with "ROOT/".

--no-strip

Don't strip files at all.
