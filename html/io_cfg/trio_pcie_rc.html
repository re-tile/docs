This document describes the following groups of registers:

  • PCIe MAC RC (0x0000-0x016c) : These registers are the PCIe configuration space for the MAC when in root complex mode.

    Base Address Register 0
    Bits Name Type Reset Description
    31:4 ADDR RW 0 BAR 0 base address bits (for a 64-bit BAR, the remaining upper address bits are in BAR 1). The BAR 0 Mask value determines which address bits are masked (e.g. this field is writable from MAC_PROTECTED space to control BAR size or from the MAC_STANDARD interface to set bar value).
    3 PREF RW 0 Indicates if the memory region is prefetchable: 0 = Non-prefetchable 1 = Prefetchable. If BAR 0 is an I/O BAR, bit 3 is the second least significant bit of the base address. Bits [3:0] are writable through the MAC_STANDARD interface.
    2:1 IS_64_BIT RW 0 Bits [2:1] determine the BAR type: 00 = 32-bit BAR 10 = 64-bit BAR. If BAR 0 is an I/O BAR, bit 2 the least significant bit of the base address and bit 1 is 0. Bits [3:0] are writable through the MAC_STANDARD interface.
    0 MEM RW 0 1 = BAR 0 is an I/O BAR Bits [3:0] are writable through the MAC_STANDARD interface.
