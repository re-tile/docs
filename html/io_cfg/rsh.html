This document describes the following groups of registers:

  • Common (0x0000-0x0038) : Each IO device contains a common set of registers used for device discovery and general setup.

  • RSH Config (0x0000-0x0c80) : The rshim contains chip-wide services for boot and diagnostics. This register set controls those services. Specific rshim-hosted devices (UART, I2C, etc.) are defined in their own register set.
